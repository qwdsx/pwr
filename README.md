## A CLI app that generates random passwords

```console
Usage: pwr gen [OPTIONS]

Options:
      --length <LENGTH>  length of password (default 16)
  -n, --numbers          boolean to NOT include numbers
  -u, --uppercase        boolean to NOT include uppercase letters
  -l, --lowercase        boolean to NOT include lowercase letters
  -s, --symbols          boolean to NOT include symbols (@#$%&=+?)
  -h, --help             Print help
```
